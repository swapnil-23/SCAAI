# SCAAI

In this project I have built an fully functional website called STOCKET where the user can the all the live equity portfolio data.
The functionality provided by the website are:
1. Access to live equity portfolio data from the NSE(National Stock Exchange Of India) 
2. Accurate market charts for the chosen equity portfolio
3. Market Basics for the beginner 
4. Market related news from all over the world
5. Provision of technical analysis dashboard where the user can have access to more than 100 technical indicators.
6. Future price prediction of the given portfolio
7. Can get the most accurate suggestion from the advance technology called Py Chat GPT.
